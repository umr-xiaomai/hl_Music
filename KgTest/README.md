# KgTest 终端播放器

`KgTest` 是仓库里的终端 TUI 播放器，不保证好用

## 运行

在仓库根目录执行：

```bash
dotnet run --project KgTest/KgTest.csproj
```

## 页面

- `每日推荐`
- `私人 FM`
- `发现歌单`
- `排行榜`
- `搜索`
- `我的歌单`
- `播放队列`
- `设置`
- `账号`


## 快捷键

| 快捷键 | 功能 |
| --- | --- |
| `←` / `→` | 切换页面；沉浸播放页中用于快退 / 快进 |
| `Tab` | 切换页面 |
| `↑` / `↓` | 移动选择 |
| `Enter` | 打开歌单 / 播放歌曲 / 调整当前设置项 |
| `Space` | 播放 / 暂停 |
| `n` / `b` | 下一首 / 上一首 |
| `[` / `]` | 快退 / 快进 10 秒 |
| `/` | 搜索 |
| `i` | 进入 / 退出沉浸播放页 |
| `+` / `-` | 普通页面调音量；设置页调整当前设置或 EQ |
| `s` | 随机播放 |
| `t` | 智能过渡开关 |
| `l` | 歌词显示模式切换 |
| `e` | 进入设置页；设置页内切换环绕 |
| `r` | 刷新当前页面 |
| `d` | 私人 FM 中标记“不喜欢”并切下一首 |
| `F5` | 二维码登录链接 |
| `F6` | 手机号验证码登录 |
| `F9` | 退出登录 |
| `q` / `Esc` | 返回 / 退出 |

## 设置页

设置页支持用 `↑` / `↓` 在所有设置项之间移动，包含顶部设置和 EQ 区域，不会只困在 EQ 部分。

- `音质`：`128` / `320` / `flac` / `high`
- `环绕`
- `智能过渡`
- `随机播放`
- `歌词模式`
- `EQ 1` 到 `EQ 10`

在设置页中：

- `Enter` 调整当前设置项。
- `+` / `-` 调整当前设置项或 EQ 增益。
